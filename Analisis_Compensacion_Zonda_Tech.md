## ⚠️ CONSIDERACIONES IMPORTANTES

### 1. Tipo de Cambio
- Utilizado 850 CLP/USD como referencial
- El sueldo se define en USD pero se paga en CLP
- Se recomienda fijar cláusula de "piso" (no baja bajo X CLP)

### 2. Bonificación
- No es garantizada, está supeditada a KPIs
- Se paga trimestralmente según avance
- Exige desempeño demostrables

### 3. Período de Prueba
- 90 días de período de prueba
- Posibilidad de ajuste salarial tras prueba
- No es tiempo mínimo para reducción de bono

### 4. Provisiones Legales
- Cumple con salario mínimo chileno
- Incluye todos los beneficios obligatorios (AFP, Salud, Seguro)
- Negociable según contratación

### 5. Benchmarking Actual
- Este análisis es referencial para 2026
- Se recomienda validar con agencias de recruitmen especializado
- Mercado tech/gaming puede fluctuar +/- 5-10%
